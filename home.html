How to use GitHub?

Following guide will be helpful in order to understand the use of GitHub

First Step: Creat repository on GitHub

In order to create repository you first have to go to repositories tab and then click new as shown in image

Second Step: Clone repository in your empty project

Following steps should be taken in order to clone the repository in your system

Copy the ssh of your repository and run the following command on your project terminal

git clone 'your repository's ssh'

Third Step: Add any file

In order to add any file in your GitHub, run the following command

git add filename

Fourth Step: Commit the added file in your local repository

In order to commit the added file, you have to run the following command

git commit -m "Any message"

Fifth Step: Push the changes

After commiting the changes, push the commited changes to your remote repository by running the following commands

git push
